# Betting App :soccer:

### Technologies used:
- ASP.NET Core 5
- PostgreSQL 12.3
- Entity Framework

-----

### How to run?

Modify `appsettings.json` with valid connection string, and run command inside `server` folder:

 ```
 dotnet run
 ```

Install npm modules and run vuejs app with command:
```
npm run serve
```
